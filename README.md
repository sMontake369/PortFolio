## Learning PortFolio(Frontend)
作成したラーニングポートフォリオのフロントエンドのみのデータ 

起動はできるが、バックエンドがいないため、ログインやポートフォリオの閲覧といった機能は使用不可

## Installation
1. npm install 必要なパッケージのインストール
2. npm dev run 起動
3. localhost:3000へアクセス
